getTemplateFromDataObject

(
  • data_object
  • form_id
  • flag
)

Defined in src\js\core\app.forms.js:631

Generates a form template from a data object. Assumes that every field in JSON data_object is of type string and thus always makes text input elements

Parameters:

  • data_object Object

    Object that contains the data of which a template is generated.

  • form_id String

    ID of the form template that is to be generated.

  • flag String

    If flag == "no_default_values", values of data_object will be ignored and not be set as default values in the template
