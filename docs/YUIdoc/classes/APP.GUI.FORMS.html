Defined in src\js\core\app.gui.forms.js:108

Creates a form on screen that the user can expand and collapse/minimize. When minimized, the user sees only the header.

Parameters:

  • parent Object

    Parent element in DOM

  • element_prefix String

    Prefix of the IDs of all DOM nodes created here.

  • Shall Boolean

    the form be expanded or not at the beginning.

  • on_expand_collapse Function

    Callback function that is called, when the user expands or collapses the form. Parameter is "true" when expanding and "false" when collapsing.

  • on_delete Function

    Callback function to be called, when the user clicks on the delete icon. Parameter "id" is passed.

  • id String

    Custom id that can be appended to the form. This ID is passed to on_delete when the user clicks on the delete icon.

Returns:

Object:

Object with several created DOM nodes. Keys are "form", "header", "label", "content", "delete_icon", "display_icon"
