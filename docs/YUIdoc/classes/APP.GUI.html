createXMLOutputDIV

  • title
  • textarea_id
  • value
  • filename
  • mockup
  • http_post_information
)

Defined in src\js\core\app.gui.js:983

Creates a GUI form that can contain an XML document. With this form, it is possible then to download the XML or to send it to an URL via HTTP POST.

Parameters:

  • parent Object

    Parent DOM element

  • title String

    Heading of the form.

  • textarea_id String

    DOM element id of the textarea, that contains the XML document.

  • value String

    XML document

  • filename String

    Name of the file that is saved on the computer, when the user clicks on "Save"

  • mockup Boolean

    If true, the form's color will be red and there will be a warning message to not use this XML document as it is invalid.

  • http_post_information Object

    Object that contains information on how to send this XML document via HTTP post. Keys are "xml_string_key" (Key of the POST parameter that contains the XML string), "url", "additional_headers" (array of key-value-pairs, that are added as headers to the HTTP request) and "additional_data" (data that is appended to post payload)

Creates a user input field where the user either can select an option from a given vocabulary or type in a custom option. As always, there is a heading above the element.

Parameters:

  • parent Object

    Parent element in DOM

  • title String

    Title or heading that is displayed above the form element

  • name String

    Name of the input elements

  • id String

    ID of the input elements. Note that both select and input will have the same ID.

  • options Object

    Array with strings. Vocabulary of values.

  • value String

    Value of the input field. Note that if the value is part of options, the select element will be used, otherwise the text input element.

  • hover String

    HTML Title Element value. This value is displayed as a tooltip when mouse is hovering above the element.

  • className String

    Class name of the select and input elements.

Returns:

Object:

Returns an object that contains both the select and the input element. Keys are "text" and "select".
