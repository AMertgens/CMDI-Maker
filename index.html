Calculate Actor's Age

When this feature is activated, CMDI Maker checks if the age of an actor (if it has not been specified already) can be calculated from the actor's birth date and the session date.
When an age can be calculated, it will appear in the output file.
(Age = Session Date - Actor's Birth Date)

CMDI Metadata Creator

The CMDI metadata format requires the name of a metadata creator. This is probably you. If so, please type in your name.


Delete Web Storage Actors Database

CMDI Maker saves all your actors in a Web Storage browser database, so that they are kept, even if you close the browser window.


Delete IndexedDB Recall Data

CMDI Maker saves the metadata form in an IndexedDB browser database, so that your input is kept, even if you close the browser window.
But if Auto Save is on, new data will be saved automatically!
