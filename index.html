CMDI Metadata Creator

The CMDI metadata format requires the name of a metadata creator. This is probably you. If so, please type in your name.


Delete Recall Data

CMDI Maker saves the metadata form in the browser. Your input is kept, even if you close the browser window.
However, if Auto Save is on, new data will be saved automatically!
This function deletes all data of the active profile!


Hard Reset

Delete all data that CMDI Maker has ever stored on your PC.


CMDI Maker

A tool for linguists for easily creating IMDI and CMDI metadata to upload to a language archive.


Part of CLASS - Cologne Language Archive Services
